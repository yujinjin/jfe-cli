# jfe-cli

前端项目脚手架

项目名称：jfe-cli

### 用法

```
jfe create <project-name>
```

### commands 指令目录

-   create 创建项目

```
// 用法
jfe create manager
```

-   download 下载项目模板

```
// 用法
jfe download 360-manager-h5
```

-   upgrade 升级 cli 脚手架

```
// 用法
jfe upgrade
```

### 不链接到全局，直接使用 npx 调试本地命令

```
npx . -v
npx . --help
npx . create test-project

```

### 可用的模板

-   360-manager-h5 后台管理系统多页项目模板
